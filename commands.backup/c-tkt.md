Notionデータベース（ID: `2e353e837a8a80bc932ce6b1e87950cb`）からチケットを削除するコマンド。

## 引数

引数によって削除対象ステータスを切り替える。

| 引数 | 削除対象ステータス |
|------|-----------------|
| なし | 「対応しない」＋「完了」 |
| `dont` | 「対応しない」のみ |
| `done` | 「完了」のみ |

## 手順

### Step 1: チケット一覧を取得して表示

引数に応じて対象ステータスを決定し、`mcp__notion__query-database` ツールでクエリする。

- database_id: `2e353e837a8a80bc932ce6b1e87950cb`
- 引数なし：`{"or": [{"property": "ステータス", "status": {"equals": "対応しない"}}, {"property": "ステータス", "status": {"equals": "完了"}}]}`
- 引数=`dont`：`{"property": "ステータス", "status": {"equals": "対応しない"}}`
- 引数=`done`：`{"property": "ステータス", "status": {"equals": "完了"}}`

取得結果を以下の形式で表示する（チケットNo昇順）:

```
以下のチケットを削除します（ステータス：対応しない・完了）  ← 対象ステータスを実際の値に合わせて表示

| No | チケット名 | カテゴリ | ステータス |
|----|-----------|---------|----------|
| T001 | チケット名 | カテゴリ名 | 対応しない |
...

合計 X 件

削除してよいですか？ (y/n)
```

### Step 2: ユーザーに確認

ユーザーの入力を待つ。

- `y` または `Y` → Step 3 へ進む
- `n` または `N` → 「削除をキャンセルしました。」と表示して終了
- その他 → 「y または n を入力してください。」と表示して再度確認

### Step 3: 削除実行

取得した全チケット（ページ）を1件ずつ `mcp__notion__delete-block` ツールで削除する。

- block_id には各チケットのページIDを使用する

全削除が完了したら以下を表示する:

```
X 件のチケットを削除しました。
```

### 注意事項

- has_more が true の場合は next_cursor を使って全件取得すること
- 削除はページ単位（page IDをblock_idとして渡す）で行う
