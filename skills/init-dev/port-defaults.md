# ポート番号デフォルト値

新規プロジェクトは既存プロジェクトと競合しないよう、デフォルト値から +1 した値を使用する。

| 変数名 | デフォルト値 | 2プロジェクト目の例 |
|--------|------------|-----------------|
| `APP_PORT` | 4431 | 4432 |
| `DB_LOCAL_PORT` | 33016 | 33017 |
| `MAILHOG_LOCAL_PORT` | 1251 | 1252 |
| `MAILHOG_WEB_PORT` | 8251 | 8252 |
| `VITE_PORT` | 51731 | 51732 |

3プロジェクト目以降はさらに +1 して競合がないか `lsof -i :{ポート番号}` で確認すること。
